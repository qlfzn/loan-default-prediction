# Loan Default Prediction
*To predict if a loan application will default or not default (default = not paying back the loan given)*

## Problem
- Most banks/financial institutions offer financial services, including money lending (in form of loans). <br>
- People can apply and receive loans from banks to help with their expenses and business capital. Loan applicants are tied to a contract and they are obligated to pay back the loan within a time period. <br>
- However, some applicants failed to pay back the loans in-time, which can hurt the banks financially and suffer losses.<br>

## Objective
This project aims to develop a good prediction model to predict whether a loan applicant will default or not, based on their historical financial and demographic data.<br><br>
The focus is to classify the potential defaulters among all applicants in the dataset.
